# XPilot

Terminal-first AI assistant for X creators.

## Features

- AI research
- GitHub research
- Reddit research
- Hacker News research
- Web search
- Tweet generation
- Thread generation
- Critique
- Local memory
- Copy generated tweets directly to clipboard

---

## Installation

```bash
git clone https://github.com/yourname/xpilot.git

cd xpilot

python -m venv .venv
```

Windows

```bash
.venv\Scripts\activate
```

Linux / Mac

```bash
source .venv/bin/activate
```

Install

```bash
pip install -e .
```

Create environment file

```bash
cp .env.example .env
```

Run

```bash
xpilot
```
